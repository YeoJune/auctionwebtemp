WMS 관리

관리자

1단계. 낙찰완료 조회 → 내부바코드 생성/출력

국내에 도착한 박스를 보고 1) 경매 진행 날짜 확인 2) 날짜/경매번호 선택 3) 낙찰완료 조회 후 전체 선택 출력하면 국내도착존으로 자동 반영됩니다.
유형 입찰ID 경매번호 아이템ID 상품명 회원사 상태 내부바코드

2단계. 스캔 처리

현재 존: 미설정 (직원모드에서는 먼저 존 코드를 스캔하세요)
입력창 클릭 후 영문(ABC) 상태에서 1) 존 바코드 스캔 2) 물건 바코드 연속 스캔

3단계. 존별 현황

숫자를 클릭하면 해당 존 물건만 아래 목록에 표시됩니다.

4단계. 최근 물건

ID 사진 경매유형 내부바코드 상품명 회원사 상태 위치 도착후경과 현재존 체류시간 갱신시각
